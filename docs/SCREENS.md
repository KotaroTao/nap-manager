- メール未認証時: 「メール認証が完了していません。[認証メールを再送信]」

### 4.1 ブレークポイント

| 名称 | 幅 | 用途 |

| `sm` | 640px | スマートフォン |
| `md` | 768px | タブレット |

| `xl` | 1280px | デスクトップ |
| `2xl` | 1536px | 大型デスクトップ |

### 4.2 モバイル対応方針

- サイドバー: ハンバーガーメニューに変換
- テーブル: カード形式に変換
- 複数カラム: シングルカラムに変換
- モーダル: フルスクリーン表示
